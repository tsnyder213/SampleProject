The Objective

The goal of this site is to demonstrate my knowledge of UI and Angular practices. This mock site was written in one day using Angular 8 with Kendo and Angular Material components. Instead of one project, I made a few different UI components with snippets of functionality. The code for this site will be on GitHub and I will send the link to find it.

Login Page

This is an interactive sign in/sign up UI. This is tied to an auth guard to protect certain routes from unauthorized access. For example, attempt to access http://tannersnyder.com/protectedpage without logging in on the UI. Now go to the UI and put in username: 'Admin', password: 'Testing123' and watch the magic happen.

Simple Dashboard

This is a simple dashboard with dummy data. It doesn't really do anything but, it looks pretty!

Error Page

Simple error handling page that will handle all incorrect URL requests.
